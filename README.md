## 🚀 **Cara Penggunaan**

### 📦 **Installation**


# 1. Clone repository
git clone https://github.com/OmhcSilence/rebuild-vps.git
cd rebuild-vps

# 2. Beri permission
chmod +x simple-rebuild.sh

# 3. Jalankan sebagai root
sudo bash simple-rebuild.sh


### 🎯 **One-Liner Installation**


# Download dan jalankan langsung
sudo bash -c "$(curl -fsSL https://raw.githubusercontent.com/OmhcSilence/vps-rebuild-script/main/simple-rebuild.sh)"


### 📊 **Cek Status VPS**


# Download status checker
curl -fsSL https://raw.githubusercontent.com/OmhcSilence/vps-rebuild-script/main/status.sh | bash

## 🐛 **Error & Solusi**

<details>
<summary><b>🔧 Klik untuk troubleshooting</b></summary>

<br/>

### Error: `syntax error near unexpected token`

**Solusi:**

# Pastikan menggunakan bash, bukan sh
sudo bash script.sh

# Atau install dos2unix untuk fix formatting
sudo apt install dos2unix
dos2unix script.sh


### Error: `Permission denied`

**Solusi:**

# Beri permission execute
chmod +x script.sh

# Jalankan sebagai root
sudo ./script.sh


### Error: `Package not found`

**Solusi:**

# Update repository terlebih dahulu
sudo apt update

# Jalankan ulang script
sudo bash script.sh


</details>

## 💡 **Tips & Trik**


# 1. Backup sebelum rebuild (opsional)
tar -czf backup-vps-$(date +%Y%m%d).tar.gz /var/www /etc/nginx /var/lib/mysql

# 2. Simpan log rebuild
sudo bash simple-rebuild.sh 2>&1 | tee rebuild.log

# 3. Cek hasil cleanup
df -h && free -h && ps aux | wc -l

# 4. Monitor realtime selama rebuild
watch -n 1 'free -h; df -h /; ps aux | wc -l'
